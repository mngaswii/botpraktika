# Чат-бот TEA — онлайн-школа иностранных языков

Чат-бот для ВКонтакте и MAX, который помогает пользователям выбрать язык, пройти тест уровня, подобрать обучение и связаться с менеджером.

## 🚀 Функционал

- Главное меню с выбором действий
- Выбор языка (английский, китайский, японский, корейский)
- Полноценный тест уровня (5 вопросов)
- Подбор обучения
- Связь с менеджером ("Позвать человека")
- Акции и сертификаты
- Запоминание состояния диалога (память пользователя)
- Поддержка двух платформ: ВКонтакте и MAX
- Веб-административная панель

## 📋 Требования

- Windows / Linux / macOS
- .NET Framework 4.8 или выше
- .NET Core SDK 6.0 или выше
- Visual Studio 2022
- Сообщество ВКонтакте
- Аккаунт разработчика MAX
